## ğŸš€ Overview
Smart Attendance System using QR & Face Recognition is a modern, AI-powered automated attendance monitoring solution. It combines QR code scanning and face recognition technology to accurately track attendance in schools, colleges, offices, and events. The system is secure, fast, and eliminates proxy attendance, providing real-time reporting and analytics

## âœ¨ Features
<b>QR Code Verification:</b> Unique QR codes for each user.

<b>AI Face Recognition:</b> Secure and accurate identity verification.

<b>Dual Authentication:</b> Attendance is recorded only when both QR and face match.

<b>Real-Time Attendance Logging:</b> Instant updates to database.

<b>Admin Dashboard:</b> View logs, analytics, and attendance reports.

<b>Cloud/Local Database Support:</b> supabase

<b>Secure & Proxy-Proof:</b> Prevents fake or duplicate attendance.
